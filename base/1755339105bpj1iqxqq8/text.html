Общий

После запуска сервера Gokapi вы можете войти в систему, используя учетные данные администратора, перейдя по адресу http(s)://your.gokapi.url/admin`


Здесь вы можете просматривать список файлов, управлять ими и загружать новые. Вы также увидите три поля:


Разрешенные загрузки позволяют вам установить, сколько раз файл может быть загружен, прежде чем он будет удален.


Срок действия в днях позволяет вам установить, через сколько дней файл будет удален последний раз.


Password позволяет установить пароль, который пользователь должен ввести перед загрузкой файла. Обратите внимание, что файл на сервере хранения не зашифрован.
