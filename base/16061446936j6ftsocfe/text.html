> Выполнение «masscan --help»
MASSCAN - это быстрый сканер портов. Основными входными параметрами являются
IP-адреса / диапазоны, которые вы хотите просканировать, и номера портов. Пример
следующее, которое сканирует сеть 10.x.x.x на наличие веб-серверов:
masscan 10.0.0.0/8 -p80
Программа автоматически определяет настройки сетевого интерфейса / адаптера. Если это
не работает, вам придется установить их вручную. Ниже приводится
пример всех необходимых параметров:
--адаптер-ip 192.168.10.123
--adapter-mac 00-11-22-33-44-55
--router-mac 66-55-44-33-22-11
Параметры можно установить либо из командной строки, либо из файла конфигурации. В
имена одинаковы для обоих. Таким образом, указанные выше настройки адаптера будут
отображаются в файле конфигурации следующим образом:
адаптер-ip = 192.168.10.123
адаптер-mac = 00-11-22-33-44-55
router-mac = 66-55-44-33-22-11
Все параметры с одним тире имеют эквивалент двойного тире,
поэтому '-p80' совпадает с '--ports 80' (или 'ports = 80' в файле конфигурации).
Чтобы использовать файл конфигурации, введите:
masscan -c <имя файла>
Чтобы сгенерировать конфигурационный файл из текущих настроек, используйте --echo
вариант. Это останавливает фактическое выполнение программы и просто повторяет
вместо этого текущая конфигурация. Это полезный способ создания
ваш первый файл конфигурации или просмотрите список параметров, которые вы не знали
около. Предлагаю вам попробовать прямо сейчас:
masscan -p1234 --echo
