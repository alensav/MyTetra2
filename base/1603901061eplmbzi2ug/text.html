Описание

  • Есть возможность произвольно выбрать папку заметок (можно использовать несколько папок заметок);
  • Имеется возможность поиска в подстроке заметок, а результаты поиска выделены в заметках;
  • Программа может работать с настраиваемыми сочетаниями клавиш;
  • Поддержка скриптов и онлайн-хранилище скриптов, где вы можете устанавливать скрипты внутри программы;
  • Расширение браузера QOwnNotes Web Companion для добавления заметок из выделенного текста и других функций: плагин для Chrome, плагин для Firefox;
  • Отслеживаются внешние изменения файлов заметок (перезагружаются заметки или список заметок);
  • Есть возможность восстановить более старые версии ваших заметок с вашего сервера ownCloud / Nextcloud, а также потерянных;
  • Различия между текущей и измененной извне заметкой отображаются в диалоговом окне;
  • Markdown подсветка заметок и режим предварительного просмотра markdown;
  • Заметки получают свое имя из первой строки текста заметки (как в веб-приложении заметок ownCloud), а текстовые файлы заметок автоматически переименовываются, если изменяется первая строка;
  • Совместим с веб-приложением заметок ownCloud и мобильными приложениями ownCloud notes;
  • Совместимость с функцией выборочной синхронизации ownCloud, поддерживая неограниченное количество папок заметок с возможностью выбора соответствующей папки на вашем сервере;
  • Управлять списками задач ownCloud (задачи ownCloud или Tasks Plus / Calendar Plus) или использовать другой сервер CalDAV для синхронизации ваших задач;
  • Есть шифрование заметок (встроен AES-256 или вы можете использовать собственные методы шифрования, такие как Keybase.io (encryption-keybase.qml) или PGP (encryption-pgp.qml));
  • Поддержка тёмной темы оформления;
  • Поддержка тем для подсветки синтаксиса markdown;
  • Все панели могут быть размещены где угодно, они могут даже плавать или складываться (полностью закрепляемый);
  • Поддержка значков темы freedesktop, вы можете использовать QOwnNotes со своими собственными значками на рабочем столе и с вашей любимой тёмной темой для рабочего стола;
  • Поддержка иерархических тегов заметок и вложенных папок заметок;
  • Поддержка обмена заметками на вашем собственном ownCloud / Nextcloud сервере;
  • Режим Vim;
  • Импорт из Evernote.

Установка

Установить посредством PPA-репозитория

Чтобы установить самую свежую стабильную версию QOwnNotes в Ubuntu, можно воспользоваться PPA репозиторием. Для этого выполните последовательно в терминале команды:

sudo add-apt-repository ppa:pbek/qownnotes
sudo apt-get update
sudo apt-get install qownnotes
