Кросс-платформенный персональный менеджер для накопления информации, который может пригодится для запоминания данных и структурирования заметок. Переведен на русский язык.


Особенности:

бесконечные ответвления в дереве заметок;

произвольная сортировка заметок в ветви;

произвольная сортировка ветвей в родительской ветви;

копирование / вставка заметок и веток;

кликабельные теги;

WYSIWYG редактор;

шифрование заметок с помощью RC5-32/12/16 PBKDF2+;

синхронизация с любой облачной системой хранения или системой контроля версий (т.е. Git на github.com);

поиск в базе;

журнал действий;

экспорт в PDF;

прикрепление файлов;

тэгирование;

вставка картинок;

импорт и экспорт веток.
