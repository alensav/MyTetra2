PRODUCT DESCRIPTION

 

Описание:

Модуль DAC представляет собой сверхдоступный высококачественный ЦАП для Raspberry Pi.

Поскольку это цифровой звук, он звучит действительно хорошо, намного лучше, чем встроенный аналоговый звук. Стереоразъем уже припаян к плате.

 

Особенность:

Линейный стереоразъем

Плата формата pHAT

Использует ЦАП PCM5102A для работы с интерфейсом Raspberry Pi I2S.

 

Спецификация:

Тип: декодер I2S DAC

Модель: ПКМ5102

 

Приложение:

A/V-ресивер

DVD, BD-плеер

HDTV ресивер

Приложения требуют аудиовыхода 2VRMS

 

Посылка включает:

1 * Модуль декодера ЦАП I2S PCM5102
