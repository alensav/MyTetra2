DOCKER

У этого образа нет версии, совместимой с вашим процессором. Вы можете попробовать использовать QEMU для его эмуляции.


https://www.stereolabs.com/docs/docker/building-arm-container-on-x86/ 34


sudo apt-get install qemu binfmt-support qemu-user-static

docker run --platform linux/amd64 ...

Таким образом, вы можете использовать версию AMD64, но эмуляция не всегда идеальна и может быть медленнее, чем запуск контейнера из совместимого образа.
