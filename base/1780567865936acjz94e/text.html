LXC

1. Создание контейнера (аналог lxc launch)

Используем шаблон download, который скачает образ из интернета:

bash

# Интерактивный режим (спросит дистрибутив, релиз и архитектуру)

sudo lxc-create -t download -n my-first-container


# Или сразу указать параметры (например, Debian 11 / Ubuntu 20.04)

sudo lxc-create -t download -n my-ubuntu -- -d ubuntu -r focal -a amd64

# Интерактивный режим (спросит дистрибутив, релиз и архитектуру)sudo lxc-create -t download -n my-first-container# Или сразу указать параметры (например, Debian 11 / Ubuntu 20.04)sudo lxc-create -t download -n my-ubuntu -- -d ubuntu -r focal -a amd64


2.2. Просмотр списка контейнеров (аналог lxc list)

bash

sudo lxc-ls -f

3. Запуск и остановка

bash

sudo lxc-start -n my-first-containersudo lxc-stop -n my-first-container

4. Вход внутрь контейнера (аналог lxc exec ... bash)

bash

sudo lxc-attach -n my-first-container

5. Удаление контейнера

bash

sudo lxc-destroy -n my-first-container
