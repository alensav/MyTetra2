Шпаргалка для повседневной работы

Быстрый доступ:

bash

# SSH в контейнер

ssh root@10.10.10.100


# Статус всех сервисов

sudo lxc-ls -f && docker ps


# Filebrowser

# Откройте:

http://localhost:8000

# SSH в контейнерssh

root@10.10.10.100


#Статус всех сервисов


sudo lxc-ls -f && docker ps

# Filebrowser# Откройте:


http://localhost:8000

Управление контейнерами:

bash

# LXC

sudo lxc-start -n ubuntu22


sudo lxc-stop -n ubuntu22


sudo lxc-attach -n ubuntu22 # Прямой вход без SSH


# Docker

docker restart filebrowser


docker logs -f filebrowser
