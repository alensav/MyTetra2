Если появится ошибка подключения к БД:

Проверьте, что MariaDB запущена:

На VPS выполните:

bash
1
sudo systemctl status mariadb

Если не запущена:

bash
1
2
sudo systemctl start mariadb

sudo systemctl enable mariadb

Проверьте, что БД создана:

bash
1
sudo mysql -u root -e "SHOW DATABASES;"

Должна быть shop_db. Если нет — создайте заново:

bash
1
2
3
4
5
6
sudo mysql -u root << EOF

CREATE DATABASE shop_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

CREATE USER 'shop_user'@'localhost' IDENTIFIED BY 'ShopPass2026!';

GRANT ALL PRIVILEGES ON shop_db.* TO 'shop_user'@'localhost';

FLUSH PRIVILEGES;

EOF


После успешного подключения:

WordPress попросит создать файл wp-config.php. Нажмите "Запустить установку".

Затем заполните основную информацию:

  • Название сайта: Магазин проектной документации
  • Имя пользователя: admin
  • Пароль: придумайте надежный (сохраните!)
  • Email: ваш email
  • ✅ Поставьте галочку "Попросить поисковые системы не индексировать сайт" (пока настраиваете)

Нажмите "Установить WordPress".
