Краткая шпаргалка: управление ВМ после настройки


Задача Команда (на хосте)


Выполнить команду внутри ВМ incus exec test-container2 -- <команда>


Подключиться к консоли incus console test-container2


Скопировать файл в ВМ incus file push ~/file.txt test-container2:/tmp/ (требует 9p или агент)


Пробросить порт (хост:ВМ) incus config device add test-container2 http80


proxy listen=tcp:0.0.0.0:8080 connect=tcp:127.0.0.1:80


Остановить / запустить incus stop test-container2 / incus start test-container2

Посмотреть логи ВМ incus info test-container2 --show-log
