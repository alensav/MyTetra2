Не работает или не открывается phpmyadmin в Ubuntu по http://localhost/phpmyadmin/

Если через http://localhost/phpmyadmin/ не открывается, добавьте в файл

/etc/apache2/apache2.conf строку

Include /etc/phpmyadmin/apache.conf после чего перезапустите апач

 

/etc/init.d/apache2 restartИ будет Вам счастье.
