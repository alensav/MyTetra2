Альт Рабочая станция

Альт Рабочая станция — дистрибутив для корпоративных рабочих мест, разработанный ООО «Базальт СПО».

Возможности

  • дистрибутив включает ОС и набор пользовательских приложений, которые можно использовать непосредственно с флэшки (например, для оценки совместимости) или установить на жёсткий диск;
  • удобная графическая среда MATE, наследующая опыт применения дистрибутивов Кентавр, СПТ и других;
  • возможность дополнительной установки приложений с того же носителя (категории «Офис», «Интернет/сети», «Мультимедиа», «Поддержка сторонних приложений», «Виртуализация», «Издательство»);
  • наличие графических средств настройки системы, включая аутентификацию (в том числе через Active Directory и LDAP/Kerberos), установку и синхронизацию времени, управление пользователями, группами, просмотр системных журналов и добавление принтеров;
  • широкий выбор различных программ для взаимодействия с корпоративной и облачной инфраструктурой, работы в сети интернет, с документами, со сложной графикой и анимацией, для обработки звука и видео, создания и запуска виртуальных машин и доступа к ним.

Программная платформа

p8 (Восьмая платформа)

Основные новшества в Восьмой платформе

  • В репозитории доступны пакеты для создания современной инфраструктуры:
    • Samba-DC — сервер контроллера домена Active Directory;
    • SOGo — сервер групповой работы с функциональностью Microsoft Exchange;
    • Proxmox-VE — система управления виртуальными машинами;
    • OpenStack — среда для создания и управления инфраструктурными облачными сервисами и облачными хранилищами;
  • Браузер Firefox собирается в трёх видах:
    • firefox — последняя версия из Sisyphus;
    • firefox-esr — последняя версия с длительной поддержкой (ESR);
    • firefox-gost — версия с патчами поддержки шифрования по ГОСТ;
  • В состав p8 включён стабильный релиз компилятора Rust.

Скачать образы

Дистрибутивы комплекта Альт Рабочая станция 8 доступны для загрузки и использования без ограничений для физических лиц; для юридических лиц необходимо приобретение лицензии или заключение лицензионного договора в письменном виде (см. лицензионный договор на операционную систему Альт Рабочая станция).

Все образы являются гибридными, то есть пригодны для записи как на DVD-диски, так и на USB-флэшки. Запись на USB-флэшку осуществляется утилитой dd в соответствии с инструкцией по записи образов (на целый диск, не раздел).

Внимание! UNetbootin, Rufus и UltraISO вместо простой записи гибридного образа ALT на флэш-накопитель портят загрузку, поэтому использование этих программ для записи образов не рекомендуется.

Изменения в 8.1

  • исправления критических уязвимостей в ядре, openssh, openssl и прочем программном обеспечении;
  • улучшена поддержка SSD и 4K-дисков;
  • автоматический запуск службы sshd;
  • в установку включён chromium;
  • добавлены различные пакеты для установки популярных сторонних приложений (*-preinstall);
  • yagf был заменён на gimagereader-qt4;
  • добавлены: VLC, libreoffice-languagetool, интерфейс доступа к удалённым графическим сеансам Connector, средство настольного поиска Recoll, openssl-engines;
  • множество мелких исправлений и улучшений.
