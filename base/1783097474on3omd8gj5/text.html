PDF_editor

=====================

Способ 1. LibreOffice (Универсальный офисный пакет)

Это самый надежный вариант для конвертации офисных документов (.docx, .xlsx, .odt, .rtf и других) в PDF.

Откройте терминал.

Обновите списки пакетов и установите LibreOffice вместе с русской локализацией:

bash

12

   sudo apt update   sudo apt install libreoffice libreoffice-l10n-ru

Как использовать: Откройте документ в LibreOffice Writer/Calc, затем выберите в меню Файл -> Экспорт в PDF.

Способ 2. Виртуальный PDF-принтер (CUPS-PDF)

Этот способ добавляет в систему виртуальный принтер. Вы сможете «распечатать» в PDF абсолютно любой документ, картинку или веб-страницу из любой программы, где есть кнопка «Печать».

В терминале выполните команду:

bash

12

sudo apt updatesudo apt install printer-driver-cups-pdf

После установки может потребоваться перезапустить систему или службу печати (sudo systemctl restart cups).

Как использовать: При печати любого документа выберите в списке принтеров Print to PDF (или CUPS-PDF). Файл обычно сохраняется в вашей домашней папке ~/PDF.

Способ 3. Консольные утилиты (Для пакетной конвертации и серверов)

Если вам нужно конвертировать документы без графического интерфейса или массово, используйте консольные инструменты.

Unoconv (для офисных документов, использует движок LibreOffice):

bash

123

sudo apt install unoconv# Пример конвертации одного файла:unoconv -f pdf документ.docx

Pandoc (для текстовых форматов, Markdown, HTML):

bash

123

sudo apt install pandoc texlive-xetex# Пример конвертации:pandoc document.md -o document.pdf
