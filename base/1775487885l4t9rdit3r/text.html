Как создать ярлык на рабочем столе в Ubuntu 24.04 (GNOME)

В Ubuntu 24.04 с рабочим окружением GNOME создание ярлыков на рабочем столе работает через .desktop-файлы. Вот несколько проверенных способов:

🔹 Способ 1: Ручное создание .desktop-файла

Создайте текстовый файл с расширением .desktop в папке ~/Desktop:

bash

1


Добавьте содержимое (пример для приложения):

ini

[Desktop Entry]

Version=1.0

Type=Application

Name=Мой скрипт

Comment=Запуск резервного копирования

Exec=/home/alensav/scripts/backup.sh

Icon=utilities-terminal

Terminal=true

12345678


🔑 Обязательные поля:

Type=Application — тип записи

Name= — отображаемое имя

Exec= — команда для запуска

📌 Полезные опции:

Icon= — имя темы иконки (например, firefox, utilities-terminal) или полный путь к файлу

Terminal=true — запускать в терминале

Path= — рабочая директория

TryExec= — проверка наличия исполняемого файла

www.glukhov.org

Сделайте файл исполняемым:

bash

chmod +x ~/Desktop/мой-ярлык.desktop

Разрешите запуск: щёлкните правой кнопкой по ярлыку на рабочем столе → «Allow Launching» (Разрешить запуск)

🔹 Способ 2: Копирование существующего .desktop-файла

Многие приложения уже имеют .desktop-файлы в системных директориях:

Откройте Files (Nautilus) → перейдите в /usr/share/applications

ubuntuhandbook.org

Найдите нужный файл (например, firefox.desktop)

Перетащите его на рабочий стол

Щёлкните правой кнопкой → Properties → включите «Allow executing file as program»

Снова правой кнопкой → «Allow Launching»

ubuntuhandbook.org

Для Flatpak-приложений файлы находятся в:

/var/lib/flatpak/app/APP_ID/current/active/export/share/applications/

🔹 Способ 3: Ярлык для веб-сайта

ini

[Desktop Entry]

Version=1.0

Type=Link

Name=Мой сайт

Comment=Открыть в браузере

URL=https://example.com

Icon=web-browser

1234567


Сохраните как ~/Desktop/сайт.desktop, сделайте исполняемым и разрешите запуск

🔹 Способ 4: Через расширение GNOME (опционально)

Расширение Add to Desktop добавляет пункт «Add to Desktop» в контекстное меню приложений:

⚠️ Для Ubuntu 24.04 (GNOME 46) расширение может требовать ручной установки из исходников:

Скачайте версию для GNOME 46 с GitHub

Распакуйте в ~/.local/share/gnome-shell/extensions/add-to-desktop@tommimon.github.com

В файле metadata.json замените "46.beta" на "46"

Перезайдите в систему и включите расширение через Extension Manager

ubuntuhandbook.org

🔹 Полезные переменные для поля Exec=

Переменная

Описание

%f

Один файл (путь)

%F

Несколько файлов

%u

Один URL

%U

Несколько URL

%c

Имя приложения

Пример: Exec=gedit %f откроет выбранный файл в редакторе

www.glukhov.org

🔧 Устранение неполадок

Проблема

Решение

Ярлык не запускается

Проверьте путь в Exec=, убедитесь, что файл исполняемый, и установлен Terminal=true при необходимости

www.glukhov.org

Нет иконки

Используйте имя темы (Icon=firefox) или полный путь к PNG/SVG

www.glukhov.org

Ярлык не виден на рабочем столе

Убедитесь, что файл в ~/Desktop, имеет расширение .desktop и разрешён к запуску

www.glukhov.org

Не отображается в меню приложений

Поместите файл в ~/.local/share/applications/ и выполните update-desktop-database ~/.local/share/applications/

www.glukhov.org
