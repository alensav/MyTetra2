Часто требуется повторять выполнение нужной команды с некоторой периодичностью, например, отслеживая изменения в её выводе.

Для автоматического повтора можно использовать утилиту watch. По умолчанию она будет через каждые 2 секунды повторять указанную команду. Например:

Здесь мы смотрим за памятью и видим изменения, которые выделяются автоматически.

watch -n 5 date

5 сек

data команда
