Загрузки MicroPython

MicroPython разработан с использованием git для управления исходным кодом, а главный репозиторий можно найти на GitHub по адресу github.com/micropython/micropython .

Полный дистрибутив исходного кода последней версии доступен для скачивания здесь:

  • микропитон-1.21.0.tar.xz (78 МБ)
  • micropython-1.21.0.zip (165 МБ)

Ежедневные снимки репозитория GitHub (не включая подмодули) доступны с этого сервера:

  • micropython-master.zip
  • pyboard-master.zip

Прошивки для различных портов и плат микроконтроллера создаются автоматически ежедневно, их можно найти ниже.
