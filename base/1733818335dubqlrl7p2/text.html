Как настроить конфигурационный файл moquitto 2 версии?

Чтобы настроить конфигурационный файл moquitto версии 2, выполните следующие шаги:


1. Установите пакеты mosquitto и mosquitto-clients:


```

sudo apt install -y mosquitto mosquitto-clients

```


2. Создайте пароль для учётной записи IoT и настройте права доступа:


```

sudo mosquitto_passwd -c /etc/mosquitto/passwd IoT

sudo chmod 777 /etc/mosquitto/passwd

```


3. Создайте конфигурацию:


```

sudo nano /etc/mosquitto/conf.d/default.conf

```


4. Вставьте следующее содержимое:


```

allow_anonymous false

password_file /etc/mosquitto/passwd

listener 1883

```


5. Сохраните файл, нажав Ctrl+O, затем Enter и Ctrl+X.


6. Перезапустите службу:


```

systemctl restart mosquitto

```


7. Проверьте работу брокера, используя команды:


```

ц

mosquitto_pub -h 192.168.50.56 -p 1883 -t "Habr" -m "Hello, Habr!" -u "IoT" -P "student"

```


Замените `192.168.50.56` на ваш IP-адрес и измените значения пароля и топика на свои.
