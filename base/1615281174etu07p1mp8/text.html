LXD/LXD command

$ Ixc image list - проверка кэша, где хранятся образы;


$ Ixc image list images:[centos] - посмотреть доступные образы конкретного дистрибутива;


$ Ixc launch images:[image fingerprint/image] [name] установить ОС из нужного имейджа по fingerprint или дистрибутиву/версии/архитектуре с указанием имени контейнера;


$ Ixc launch images:[image tingerprint/image] [name] —vm - то же, для виртуальных машин;


$ Ixc image info [image fingerprint/image] - полная информация об имейдже;


$ Ixc image show [image fingerprint/image] - чуть меньше информации;


$ Ixc image delete [image fingerprint/image] - удаление имейджа


$ Ixc list - вывод списка контейнеров;


$ Ixc start [name] - запуск контейнера;


$ Ixc stop [name] - остановка контейнера;


$ Ixc restart [name] - перезапуск контейнера;


$ Ixc delete [name] - удалить контейнер;


$ Ixc move [name] [name] - переименовать контейнер;


$ Ixc copy [name] [name] - клонировать контейнер;


$ Ixc config show [name] - вывести конфиг;


$ Ixc exec [name] [user] [command] - выполнить команду в контейнере;


$ Ixc console [name] - вывести консоль контейнера


$ Ixc file push [from] [container/directory] - копировать файл в контейнер;

$ Ixc file pull [container/directory] [to] - копировать файл из контейнера
