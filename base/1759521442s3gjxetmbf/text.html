Как искать репозитории с помощью gh?

GitHub CLI позволяет искать репозитории прямо из терминала:

bash

1

gh repo search <ключевые слова>

Например:

bash

1

gh repo search python scripts

Это покажет список репозиториев на GitHub, соответствующих запросу "python scripts".

Можно уточнить поиск с помощью флагов:

  • --language python — только репозитории на Python
  • --sort stars — сортировка по количеству звёзд
  • --limit 10 — показать первые 10 результатов

Пример:

bash

1

gh repo search useful python scripts --language python --sort stars --limit 5


Установка GitHub CLI (если ещё не установлена)

  • macOS: brew install gh
  • Windows: winget install GitHub.cli или через официальный сайт
  • Linux: см. инструкции на https://cli.github.com/

После установки выполните:

bash

1

gh auth login

чтобы авторизоваться (если нужно клонировать приватные репозитории или делать push).
