Чтобы позволить pip выбирать, какое колесо загрузить, вы можете указать ему базовый URL-адрес с помощью -fплюс имя пакета (или имя == версия), как обычные команды pip. Что-то вроде этого:

pip install -U --pre -f https://wxpython.org/Phoenix/snapshot-builds  wxPython

При этом будет получен правильный пакет в зависимости от платформы, архитектуры и версии Python, если имеется доступное совпадение.
