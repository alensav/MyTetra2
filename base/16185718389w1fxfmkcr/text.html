В данном случае для ссылки на скачивание можно использовать

какое-нибудь текстовое поле из формы сведений о товаре, например, поле

"Количество на складе (свободный текст)" (в базе данных это

quantity_txt).


Пример с использованием этого поля.

В файле inc/pm_modules.php найдите function payment_notifi_shopper

и после строк

if(! $order_data['order']['orderid']){

return false;

}

добавьте код:

global $order_id;

$order_id = $order_data['order']['orderid'];


В файле mail_tpl/payment_notifi_shopper.tpl в любое удобное место

вставьте код:

<?php

global $db, $order_id;

$table = DB_PREFIX . 'orders_items';

$result = $db->query("SELECT `itemid` FROM `$table` WHERE `orderid` = '$order_id'") or die($db->error());

$itemsids = '';

while($row = $db->fetch_assoc($result)){

$itemsids .= $row['itemid'] . ',';

}

$itemsids = substr($itemsids, 0, strlen($itemsids) - 1);

$table = DB_PREFIX . 'items';

$result = $db->query("SELECT * FROM `$table` WHERE `itemid` IN($itemsids)") or die($db->error());

while($row = $db->fetch_assoc($result)){

echo "Ссылка для скачивания \"$row[title]\": $row[quantity_txt]\n";

}

?>


Чтобы ссылки для скачивания не отображались на странице товара, в

файле дизайна design/имя-дизайна/tpl/product_detail.tpl удалите метку {quantity_txt}
