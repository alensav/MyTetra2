ThingSpeak - это облачная платформа интернета вещей (IoT), которая позволяет собирать, отображать и анализировать потоковые данные. Вы можете отправлять данные в ThingSpeak с различных устройств, настраивать их отображение в реальном времени и отправлять уведомления через Twitter и Twilio.

Аналитика MATLAB внутри ThingSpeak позволяет писать и запускать MATLAB-код для выполнения предобработки, визуализации и анализа данных. ThingSpeak позволяет инженерам и ученым прототипировать и строить IoT-системы без настройки серверов и разработки веб-приложений.

Ключевые особенности

  • Настройка устройств и отправка данных в ThingSpeak через REST API или MQTT
  • Сбор данных по запросу с устройств и других источников
  • Отображение в реальном времени текущих и исторических данных
  • Предобработка и анализ собранных данных с помощью встроенных функций MATLAB
  • Запуск аналитики автоматически по таймеру или наступлению события
  • Взаимодействие с сервисом отправки сообщений Twilio и соцсетью Twitter

Сбор данных

Отправляйте данные с датчиков в облако

Сенсоры повсюду - в наших домах, смартфонах, автомобилях, среди городской инфраструктуры и промышленного оборудования. Они измеряют информацию совершенно разных видов, например, температуру, влажность, давление, уровень шума… И эту информацию они преобразуют в числа, которые надо собирать, хранить и обрабатывать.

Почему для сбора данных стоит использовать ThingSpeak?

Датчики, или “вещи” обычно работают локально. ThingSpeak позволяет им, а также другим устройствам и даже вебсайтам хранить данные в облаке - в закрытых или публичных каналах. По умолчанию каналы ThingSpeak закрыты от других пользователей, но при желании вы можете делать их доступными для всех. Как только данные попали в канал ThingSpeak, вы можете их визуализировать, анализировать и преобразовывать. Кроме того, из ThingSpeak вы можете взаимодействовать с соцсетями, вебсервисами и различными устройствами.

Анализ

Анализируйте и отображайте данные с MATLAB

Хранение данных в облаке обеспечивает быстрый доступ к ним. Использование аналитических онлайн-инструментов позволяет вам исследовать и визуализировать данные, находить взаимосвязи, паттерны и тренды. Вы можете создавать новые данные из имеющихся, а также строить всевозможные графики.

Почему стоит использовать ThingSpeak для анализа и визуализации данных?

ThingSpeak обеспечивает доступ к функциям MATLAB, которые позволяют:

  • Обрабатывать, объединять, и преобразовывать данные
  • Выполнять вычисления по заданному расписанию
  • Визуально исследовать взаимосвязи в данных с помощью встроенных графиков
  • Объединять данные из нескольких каналов для проведения более сложного анализа

Действия

Реакция на данные

Действия на основе данных могут быть примитивными, вроде написания твита, если температура поднялась выше 30 градусов. Или вы можете сделать что-то посложнее - например, включать насос, когда вода в баке упала ниже порогового значения. Контролируйте удаленно устройства, например, электронные дверные замки, с помощью встроенного приложения TalkBack.

Почему для автоматизации действий надо использовать ThingSpeak?

ThingSpeak предоставляет возможности по взаимодействию с устройствами. Вы можете:

  • Реагировать на данные (не важно, на сырые или на обработанные) сразу при их поступлении в канал
  • Создавать очереди из команд для их выполнения на устройствах
