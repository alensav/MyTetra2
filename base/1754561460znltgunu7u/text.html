ENV


# 1. Создаём проект

mkdir my_web_app

cd my_web_app


# 2. Создаём виртуальную среду

python3 -m venv .venv


# 3. Активируем

source .venv/bin/activate


# 4. Обновляем pip

pip install --upgrade pip


# 5. Устанавливаем пакет

pip install requests


# 6. Сохраняем зависимости

pip freeze > requirements.txt


# 7. Работаем...


# 8. Выходим

deactivate

--------------------------------------------------------------------------------------------

Подсказка: автоматизация

Можно добавить алиас в ~/.bashrc:


bash

1 alias activate='source .venv/bin/activate'

Теперь в папке с .venv можно просто:


1 activate
