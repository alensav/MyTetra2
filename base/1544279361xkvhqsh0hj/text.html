===========================

Закрываем приложение из терминала

Если вам известно название приложения, то один из самых быстрых и очевидных способов - это выполнить в терминале команду:

$ killall название-программы

После выполнения этой команды, зависшее приложение должно остановиться.

Если вы не можете вспомнить как именно называется процесс программы, или после выполнения предыдущей команды система выводит, что такого процесса не существует, то установите приложение htop:

$ sudo apt-get install htop

С помощью него вы можете найти зависшее приложение из списка и остановить его. Для этого запускайте в терминале htop и ищите приложение из списка. Как только нашли его - наведите на него курсором мыши, кликните один раз и нажмите клавишу F9.
