Как в UBUNTU добавить программу в автозагрузку

Если вам понадобилось, что бы программа (любой скрипт) автоматически включалась при включении Ubuntu - ее нужно добавить в автозагрузку ОС. Делается это легко :)

Чтобы добавить программу или скрипт в автозагрузку Ubuntu нужно сделать следующее:
Создаем файл в /etc/init.d/

sudo nano /etc/init.d/local

В начало скрипта добавляем:

#! /bin/sh

Делаем так, чтобы скрипт /etc/init.d/local можно было исполнять

sudo chmod +x /etc/init.d/local

Добавляем скрипт /etc/init.d/local в автозагрузку

sudo update-rc.d local defaults 80

Все. Теперь можно добавлять в наш скрипт команды, которые должны быть выполнены при запуске системы.

Если, вдруг, понадобится удалить скрипт из автозагрузки нужно выполнить следующую команду:

sudo update-rc.d -f /etc/init.d/local remove
