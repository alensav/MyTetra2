Руководство пользователя ESP-AT

[中文]


Это документация для ESP-AT Development Framework. Проект ESP-AT был запущен и поддерживается Espressif Systems как официальный проект для SoC серий ESP32, ESP32-C2, ESP32-C3, ESP32-C6 и ESP32-S2, предоставляемых для Windows, Linux и macOS.


В этом документе описывается использование ESP-AT с ESP32 SoC . Чтобы переключиться на другой целевой SoC, выберите целевой объект из раскрывающегося списка в левом верхнем углу.
